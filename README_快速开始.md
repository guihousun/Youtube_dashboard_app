# YouTube 公开数据看板快速开始

## 1. 项目定位

这是一个 **纯公开数据版 YouTube 多频道看板**。

特点：

- 不登录 YouTube 账号
- 不使用 OAuth
- 只使用 `YouTube Data API v3` 的公开数据
- 支持多个公开频道统一管理
- 支持为频道设置别名，例如“员工1”

## 2. 你需要准备什么

只需要：

- Python
- Node.js / npx
- 一个 `YouTube Data API v3` 的 `API Key`

不需要：

- `Client ID`
- `Client Secret`
- 频道登录授权

## 3. 首次使用

双击：

- `一键配置环境.bat`

脚本会自动：

- 检查 Python / Node.js / npx
- 创建 `output/playwright/`
- 创建 `snapshots/`
- 创建 `.env.example`
- 如果缺少 `.env`，自动生成
- 如果缺少 `accounts.json`，自动生成空配置

脚本会提示输入：

- `YT_API_KEY`

## 4. 启动看板

双击：

- `启动看板.bat`

浏览器会自动打开：

- `http://127.0.0.1:8130/`

## 5. 添加频道

在页面底部 `频道管理` 中：

1. 输入公开账号名，例如 `@GolemAnton`
2. 可选输入别名，例如 `员工1`
3. 点击 `添加账号`

## 6. 看板里能看到什么

当前保留的老板视角核心内容：

- 频道总播放量较昨日快照增量排行
- 频道当前订阅数排行
- 昨日发布视频当前总播放排行
- 今日发布视频汇总
- 昨日发布视频封面墙
- 频道公开画像总览
- 频道管理

## 7. 一个重要说明

`01 / 频道总播放量较昨日快照增量排行`  
依赖本地快照。

也就是说：

- 没有昨日快照的频道不参与排行
- 会显示 `新接入 / 无昨日基准`
- 最好每天固定运行一次，让本地形成快照基准

## 8. 建议交付方式

建议直接交付整个项目目录，不要拆散文件。  
对方收到后按以下顺序即可：

1. 双击 `一键配置环境.bat`
2. 输入 `YT_API_KEY`
3. 双击 `启动看板.bat`
4. 在页面中添加公开频道

## 9. 重要文件

- `youtube_dashboard_app.py`：本地服务端
- `dashboard-prototype.html`：看板页面
- `setup_project.py`：初始化脚本
- `.env.example`：公开数据 API key 模板
- `accounts.json`：频道配置
- `snapshots/`：每日快照
