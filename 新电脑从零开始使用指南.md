# 新电脑从零开始使用指南

这份指南适用于：

- 一台新的 Windows 电脑
- 电脑上还没有 Python / Node.js
- 你已经拿到了这个项目目录

## 一、先安装 Python

1. 打开 [Python 官网](https://www.python.org/downloads/windows/)
2. 下载 Python 3
3. 安装时勾选：
   - `Add Python to PATH`
4. 安装完成后打开 `PowerShell`
5. 运行：

```powershell
python --version
```

## 二、再安装 Node.js

1. 打开 [Node.js 官网](https://nodejs.org/)
2. 下载 `LTS` 版本
3. 正常安装即可
4. 安装完成后运行：

```powershell
node --version
npx --version
```

## 三、进入项目目录

假设你把项目放在：

```text
D:\YoutubeDashboardApp
```

直接打开该目录即可。

## 四、第一次配置

双击：

- `一键配置环境.bat`

脚本会自动：

- 检查 Python
- 检查 Node.js / npx
- 创建 `output/playwright/`
- 创建 `snapshots/`
- 创建 `.env.example`
- 如有需要创建 `.env`
- 如有需要创建 `accounts.json`

然后只需要输入：

- `YT_API_KEY`

## 五、启动看板

双击：

- `启动看板.bat`

浏览器会自动打开：

- `http://127.0.0.1:8130/`

## 六、如何添加频道

在页面底部 `频道管理` 中：

1. 输入公开频道账号名，例如 `@GolemAnton`
2. 可选输入别名，例如 `员工1`
3. 点击 `添加账号`

## 七、关于“昨日快照增量”

`01 / 频道总播放量较昨日快照增量排行`

不是 YouTube 官方直接给的数据，而是：

```text
今天总播放量 - 昨日快照总播放量
```

所以：

- 新接入频道第一天不参与 01 排行
- 会显示 `新接入 / 无昨日基准`
- 建议每天至少运行一次

## 八、如果页面打不开

先检查：

```powershell
python --version
node --version
npx --version
```

再检查 `8130` 端口：

```powershell
Get-NetTCPConnection -LocalPort 8130 -State Listen
```

## 九、如果需要重置配置

可以手动编辑：

- `accounts.json`
- `.env`

然后重新运行：

- `启动看板.bat`
